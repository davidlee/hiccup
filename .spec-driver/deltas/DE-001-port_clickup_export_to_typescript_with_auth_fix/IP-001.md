## 1. Summary

- **Delta**: DE-001 – Port clickup-export to TypeScript with auth fix
- **Desired Outcome**: Working TypeScript CLI that exports ClickUp task hierarchies to JSON, handling permission errors gracefully

## 2. Context & Constraints

- **Current Behaviour**: Python script aborts on permission errors, no pagination, redundant API calls
- **Target Behaviour**: TypeScript CLI with paginated fetching, tree assembly from flat list, per-request error boundaries
- **Dependencies**: None
- **Constraints**: None — no users, no backwards compat
